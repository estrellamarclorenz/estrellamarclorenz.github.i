1. Install XAMPP
2. Start Apache and MySQL
3. Open phpMyAdmin
4. Create database named voting_system
5. Import voting_system.sql
6. Go to https://dev.mysql.com/downloads/connector/j/
7. Click “Select operating system”
8. Select “Platform Independent” and download it then unzip it
9.Open the NetBeans project
10. Click File  -> Import Project -> From Zip -> then select the “VotingSystem (database).zip” file na naa sa gdrive
11. Right click the project -> then properties -> go to libraries -> then sa right side naay “Add JAR/Folder” i click na
10. After that you should be prompted to select a file so find the extracted msql connector, it should be on the downloads folder of your pc inside a folder called “mysql-connector-j-9.7.0”, if you find it open it until you see the “mysql-connector-j-9.7.0.jar” file, click that and then click open
 11. Finally, run the project
